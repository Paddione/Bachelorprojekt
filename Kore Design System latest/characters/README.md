# Arena · Normal-people character cast

The user asked for **3 gameplay-character mocks that just resemble normal people** — a few suggestions across male / female, young / old, long / short hair, each with a randomly generated stereotype.

This page generates 3 characters at a time. Click any card to reroll it; click **reroll all** to redraw the cast. Stereotypes are drawn from a small curated list — they describe an archetype the character could be, not a class with mechanics.

> The portraits are CSS/SVG placeholders in the Arena iso-render style (60° iso, warm key + cool fill + rim, pixel-aligned silhouettes). They are **mocks for the asset pipeline** — not finished renders. If you like a direction, the next step is a Blender pass to match the existing Mage / Rogue / Tank / Warrior / Zombie sheet (`arena/assets/renders/characters/*`).

Open `index.html`.
