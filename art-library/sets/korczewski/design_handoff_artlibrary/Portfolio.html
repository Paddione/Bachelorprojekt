[ 01 — Marke ]

Eine Wortmarke. Ein Akzent. Drei Stimmen.

SERIF · SANS · MONO
[ 02 — Figuren ]

Vier Silhouetten. Keine Überschneidung.

PORTRAIT · TOKEN · PALETTE

Jede Figur ist als Silhouette erkennbar — fließendes Haar, gehörnter Helm, Dreispitz, kahle Kapuze. Erst danach kommt die Farbe. Elara trägt das grüne Kleid und das rote Haar, das du dir gewünscht hast; die anderen drei sind so weit wie möglich von ihr entfernt entworfen.

[ 03 — Brett ]

Auf dem Holz.

VORSCHAU · 1×4

Wie die Figuren auf dem Brett aussehen — als Ersatz für die vier Grundformen (Figur, Kegel, Säule, Raute) deines aktuellen Prototypen.

[ 04 — Palette ]

Zwölf Farben, jede mit einer Aufgabe.

3D / 2D · GEMEINSAM

Die Palette ist gleichzeitig die Maler-Liste für 3D-Materialien und das Vokabular für UI und Druck. Jede Farbe hat eine feste Rolle — keine Dekoration ohne Grund.

[ 05 — Requisiten ]

Kleine Dinge, die man auf das Brett legt.

PLATZHALTER · 6 STÜCK

Token-Platzhalter für Items und Marker. Stilistisch im selben Vokabular wie die Figuren — wenig Detail, klare Silhouette, eine Lichtseite.

[ 06 — Gelände ]

Sechs Materialien für das Brett.

KACHEL · 120×80

Material-Kacheln als Stimmungsproben — generiert, austauschbar. Sie zeigen das Verhältnis aus Farbe, Rauschen und Detail, das die späteren echten Texturen treffen sollen.
