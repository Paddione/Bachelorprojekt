## Pipeline Stages

Each requirement passes through five pipeline stages, tracked in `bachelorprojekt.pipeline`:

| Stage | Description |
|-------|-------------|
| `idea` | Requirement defined and understood |
| `implementation` | Code/infrastructure changes made |
| `testing` | Automated and/or manual tests passing |
| `documentation` | Documentation updated |
| `archive` | Requirement fully completed and archived |
