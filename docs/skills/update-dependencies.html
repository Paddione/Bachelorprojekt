Update Dependencies

Systematische Aktualisierung von Paketen und Frameworks: Deprecation-Audit, Major-Bumps und EOL-Check über alle Workspaces hinweg.

Auslöser: Veraltete Pakete, Sicherheits-Warnungen (Audit) oder Deprecation-Warnings

1 Audit Outdated & Deprecated

Ermittle veraltete Pakete pro Workspace:

npm outdated --long
pnpm outdated

Identifiziere Deprecations in der Tiefe:

npm audit 2>&1 | grep -i deprecat
2 Safe Updates Patch & Minor

Nutze npm-check-updates für kontrollierte Updates:

npx npm-check-updates --target minor -u && npm install

Führe nach jedem Workspace-Update die Tests aus.

3 Major Bumps Breaking Changes

Besondere Aufmerksamkeit bei:

  • Astro: Integrations (@astrojs/*) müssen mitziehen.
  • Tailwind: v4 Config-Migration.
  • Playwright: Engine-Update erforderlich (npx playwright install).
4 Verifizierung Clean Output

Stelle sicher, dass keine Deprecation-Warnings mehr auftreten:

npm install 2>&1 | grep -ic deprecat
