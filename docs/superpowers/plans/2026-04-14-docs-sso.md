# Docs SSO Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace BasicAuth on docs.korczewski.de and docs.mentolder.de with Keycloak OIDC SSO via an oauth2-proxy, matching the Invoice Ninja pattern.

**Architecture:** An `oauth2-proxy-docs` sidecar deployment sits in front of the static docs server. The Traefik ingress routes traffic to the proxy (port 4180); the proxy authenticates users against Keycloak and forwards authenticated requests to `docs:80`. The `docs` OIDC client is registered in all three Keycloak realm JSONs (dev, korczewski, mentolder).

**Tech Stack:** oauth2-proxy v7.9.0, Keycloak OIDC, Kustomize, Traefik

## Task 1: Create base oauth2-proxy-docs deployment

**Files:**
- Create: `k3d/oauth2-proxy-docs.yaml`

- [ ] **Step 1: Create the file**

```yaml
# ═══════════════════════════════════════════════════════════════════
# OAuth2 Proxy — Keycloak SSO gateway for Docs
# Authenticates users via OIDC, then forwards to docs:80
# ═══════════════════════════════════════════════════════════════════
apiVersion: apps/v1
kind: Deployment
metadata:
  name: oauth2-proxy-docs
  labels:
    app: oauth2-proxy-docs
spec:
  replicas: 1
  selector:
    matchLabels:
      app: oauth2-proxy-docs
  template:
    metadata:
      labels:
        app: oauth2-proxy-docs
    spec:
      securityContext:
        runAsNonRoot: true
        runAsUser: 65534
        seccompProfile:
          type: RuntimeDefault
      containers:
        - name: oauth2-proxy
          image: quay.io/oauth2-proxy/oauth2-proxy:v7.9.0
          args:
            - --provider=keycloak-oidc
            - --client-id=docs
            - --client-secret=$(DOCS_OIDC_SECRET)
            - --redirect-url=http://docs.localhost/oauth2/callback
            - --oidc-issuer-url=http://keycloak:8080/realms/workspace
            - --ssl-insecure-skip-verify=true
            - --skip-oidc-discovery=true
            - --login-url=http://auth.localhost/realms/workspace/protocol/openid-connect/auth
            - --redeem-url=http://keycloak:8080/realms/workspace/protocol/openid-connect/token
            - --oidc-jwks-url=http://keycloak:8080/realms/workspace/protocol/openid-connect/certs
            - --profile-url=http://keycloak:8080/realms/workspace/protocol/openid-connect/userinfo
            - --upstream=http://docs:80
            - --http-address=0.0.0.0:4180
            - --cookie-secret=$(OAUTH2_PROXY_COOKIE_SECRET)
            - --cookie-secure=false
            - --cookie-name=_oauth2_proxy_docs
            - --email-domain=*
            - --pass-access-token=true
            - --pass-authorization-header=true
            - --set-xauthrequest=true
            - --skip-provider-button=true
            - --code-challenge-method=S256
            - --insecure-oidc-allow-unverified-email=true
            - --oidc-extra-audience=docs
            - --scope=openid email profile
          env:
            - name: DOCS_OIDC_SECRET
              valueFrom:
                secretKeyRef:
                  name: workspace-secrets
                  key: DOCS_OIDC_SECRET
            - name: OAUTH2_PROXY_COOKIE_SECRET
              valueFrom:
                secretKeyRef:
                  name: workspace-secrets
                  key: OAUTH2_PROXY_COOKIE_SECRET
          securityContext:
            allowPrivilegeEscalation: false
            readOnlyRootFilesystem: true
            runAsNonRoot: true
            runAsUser: 65534
            capabilities:
              drop: [ALL]
            seccompProfile:
              type: RuntimeDefault
          ports:
            - containerPort: 4180
          readinessProbe:
            httpGet:
              path: /ping
              port: 4180
            initialDelaySeconds: 5
            periodSeconds: 10
          livenessProbe:
            httpGet:
              path: /ping
              port: 4180
            initialDelaySeconds: 10
            periodSeconds: 30
          resources:
            requests:
              memory: 64Mi
              cpu: "50m"
            limits:
              memory: 128Mi
              cpu: "200m"
---
apiVersion: v1
kind: Service
metadata:
  name: oauth2-proxy-docs
spec:
  selector:
    app: oauth2-proxy-docs
  ports:
    - port: 4180
      targetPort: 4180
```
