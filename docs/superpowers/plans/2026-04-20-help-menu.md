# Help Menu Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Kontextsensitives Slide-over Hilfemenü für Portal- und Admin-Kontext, ausgelöst per Floating-`?`-Button links unten.

**Architecture:** Eine gemeinsame `HelpPanel.svelte` Komponente (Svelte 5 Runes) empfängt `section` + `context` als Props und rendert passenden Inhalt aus einem zentralen `helpContent.ts`-Objekt. Portal- und Admin-Layout binden beide denselben Button + dasselbe Panel ein.

**Tech Stack:** Astro 5, Svelte 5 (Runes), TypeScript, CSS Custom Properties (`--ink-*`, `--fg`, `--line`, `--brass`)

<!-- website/src/components/HelpPanel.svelte -->
<script lang="ts">
  import { helpContent } from '../lib/helpContent';
  import type { HelpContext } from '../lib/helpContent';

  let {
    section,
    context,
  }: {
    section: string;
    context: HelpContext;
  } = $props();

  let open = $state(false);

  const content = $derived(helpContent[context]?.[section] ?? null);

  function toggle() { open = !open; }
  function close() { open = false; }

  function onBackdropKeydown(e: KeyboardEvent) {
    if (e.key === 'Enter' || e.key === ' ') close();
  }
</script>
