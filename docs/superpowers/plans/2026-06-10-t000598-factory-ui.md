# Plan: T000598 — Factory UI — FactoryFloor Redesign (Förderband)

## Ziel

`FactoryFloor.svelte` vollständig auf Industrial/Loft-Ästhetik umstellen: Förderband-Visualisierung mit 6 horizontalen Workstations, WorkpieceCards (160×80 px, LED-Strip, Spotlight), Monospace Detail-Panel mit Phase-Dot-Chain, und voller Mobile-Funktionsparität via Tab-Bar + Swipe. Alle bestehenden APIs und `data-testid`-Attribute bleiben erhalten.

## Tech-Stack

| Technologie | Verwendung |
|------------|-----------|
| **Svelte 5** (Runes-API: `$state`, `$props`, `$derived`) | Alle neuen Komponenten |
| **CSS Custom Properties** | Design-Tokens in `factory-tokens.css` |
| **CSS `box-shadow` / `filter: drop-shadow`** | Spotlight-Glow-Effekt |

| **Touch Events** (`touchstart`/`touchend`) | Swipe-Navigation Mobile |

| **JetBrains Mono / ui-monospace** | Monospace-Schriftstack |
