# Plan: T000704 — Brett Board-Export PNG & PDF (Qualitäts- und UX-Verbesserungen)

## Ziel

Die bestehende Export-Infrastruktur (T000466/T000605) um vier UX- und
Qualitätsverbesserungen ergänzen:

- **P1** HiDPI-PNG via `snapshot2x()` (2× Pixel-Ratio für scharfe Präsentationsexporte)
- **P2** Export-Toast-Feedback (Erfolg/Fehler, 2,5 s, analog `late-join-toast.ts`)
- **P3** Session-Code im Dateinamen (`brett-YYYY-MM-DD-{code}.png/.pdf/.json`)
- **P4** PDF Seitenumbruch bei >8-zeiliger Figurenliste (kein Überlauf bei 48 Figuren)

Kein Server-Roundtrip, kein neues npm-Dependency, kein Eingriff in geteilte
Manifeste/Konfigurationen.

### Task 3 — snapshot2x + _filename in export.ts (P1, P3)

**Datei:** `brett/src/client/ui/export.ts` (248 → ~270 Zeilen)

**P1 — HiDPI-Snapshot:**

Füge `snapshot2x` hinzu. Die Funktion benötigt `renderer`, `scene`, `camera` aus
`scene.ts`. Da `initExportButtons` bereits auf `renderer.domElement` zugreift,
wird die Signatur um `renderer`-Referenz erweitert. Import aus `scene.ts` über
`getScene()` (bereits `export`-iert).

```ts
import { getScene } from '../scene.js';

export async function snapshot2x(): Promise<string> {
  const { renderer, scene, camera } = getScene();
  const original = renderer.getPixelRatio();
  renderer.setPixelRatio(2);
  renderer.render(scene, camera);
  const dataUrl = renderer.domElement.toDataURL('image/png');
  renderer.setPixelRatio(original);
  renderer.render(scene, camera);  // Viewer-Frame wiederherstellen
  return dataUrl;
}
```

**P3 — Session-Code im Dateinamen:**

Erweitere `_isoDate()` zu `_filename(ext: string): string`:

```ts
function _filename(ext: string): string {
  const date = new Date().toISOString().slice(0, 10);
  const code = _cache.sessionCode;
  return code ? `brett-${date}-${code}.${ext}` : `brett-${date}.${ext}`;
}
```

Ersetze alle `brett-${_isoDate()}.png` / `.pdf` / `.json` durch `_filename('png')` etc.

- [ ] `snapshot2x()` in `export.ts` hinzufügen
- [ ] `_isoDate()` durch `_filename(ext)` ersetzen (alle drei Download-Funktionen)
- [ ] Import `getScene` aus `../scene.js` ergänzen
