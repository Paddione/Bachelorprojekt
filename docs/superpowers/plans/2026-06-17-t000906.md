# Direkter Datei-Share-Link im Chat (Nextcloud → Chat) — Implementation Plan [T000906]

Spec: `docs/superpowers/specs/2026-06-17-t000906-design.md`

## Ziel

Aus dem plattform-eigenen Chat (ChatRoom) heraus einen Nextcloud-Share-Link erzeugen
(intern/öffentlich) und als Chat-Nachricht einfügen. Admin **und** Kunde dürfen teilen;
ein serverseitiger Pfad-Präfix-Guard erzwingt „kein Rechte-Bypass" (Kunde nur eigener
Client-Ordner). Brand-isoliert über die vorhandene env-Resolution.

## Architektur / Tech-Stack

- **Astro 5 + Svelte 5 (Runes)** Website. Chat-System = `chat_messages`/`chat_rooms`,
  gerendert über `ChatRoomPanel.svelte`, REST unter `/api/portal/rooms/[id]/*` (Kunde) und
  `/api/admin/rooms/[id]/*` (Admin).
- **Neues pures Helper-Modul** `website/src/lib/nextcloud-shares.ts` — OCS Share-API-Client
  + Pfad-Guard. Re-Use von `nextcloud-files.ts` (Admin-Creds, `getClientFolderPath`,
  Pfad-Normalisierung). Kein DB-/API-Import (S2).
- **Neue Endpoints** `POST /api/portal/rooms/[id]/share.ts` + `POST /api/admin/rooms/[id]/share.ts`,
  die Session/Membership-Guard + `assertPathAllowed` + `createShareLink` + `addRoomMessage`
  komponieren.
- **Brand-Isolation:** `NEXTCLOUD_URL` / `NEXTCLOUD_EXTERNAL_URL` aus env/ConfigMap — keine
  Domain-Literale im Code (S3).
- **Tests:** Vitest (`fetch`-Mock) für Guard + OCS-Body; bestehende Test-Suiten erweitern.

## Risiken / Offene Punkte

- **Owner-Semantik:** Shares werden technisch unter dem Admin-NC-Account erzeugt (kein
  App-Passwort-Flow). Der Pfad-Guard garantiert, dass ein Kunde nur Eigenes teilt →
  permissions-äquivalent, dokumentierter Trade-off (Spec §3). Echte Impersonation = Folge-Ticket.
- **Such-UX v1:** nur Pfad-Eingabe + flache Namensfilterung im Scope-Ordner; Tree-Navigation
  bewusst out-of-scope.
- **`ChatRoomPanel.svelte`-Budget:** 169 Zeilen frei — bei Überschreitung Extraktion nach
  `ShareFileInline.svelte` (im Task 4 vorgesehen).
- **`messaging-db.ts`-Budget:** nur 69 Zeilen frei → **keine** neue DB-Logik dort; striktes
  Re-Use von `addRoomMessage`.
