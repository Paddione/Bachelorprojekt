# Mentolder — Print & Document Kit

Warm-paper documents that share the editorial brass-on-ink voice of the website but switch to a print-friendly palette (`--paper`, `--paper-ink`, brass `#A8823A`).

## Files
- `invoice.html` — Rechnung (A4) with line items, totals, §19 UStG note, footer
- `contract.html` — Coaching-Vertrag (A4) with numbered clauses, signature block
- `questionnaire.html` — Vorbereitungs-Fragebogen (A4) with multi-choice, scale, free-text
- `newsletter.html` — Email-style monthly brief (640px) with hero, cards, quote
- `print.css` — Shared paper styles imported by each document

All documents reuse the brand mark from `assets/logo-mark.png` and pull tokens from `colors_and_type.css`.
