# Mentolder — Website UI Kit

Live web components used across the production website.

## What's in here
- **Navigation** — top bar with brand, links, primary CTA
- **Buttons** — primary (brass pill), secondary (ghost outline), inline links
- **Forms** — contact form, newsletter inline-input, custom checkbox
- **Cards** — offer cards (with optional brass accent variant)
- **Badges** — neutral, brass, sage, ghost variants
- **Quote / testimonial** — full and compact, with brass quote glyph
- **Stats** — large serif numerals with brass emphasis
- **Lists & meta** — bulleted lists, badge clusters, meta blocks
- **Steps** — numbered editorial process
- **FAQ** — `<details>` accordion with brass chevron
- **Footer** — three-column with brand, links, bottom bar

## Files
- `index.html` — visual showcase of every component
- `kit.css` — component CSS (imports `colors_and_type.css`)

All components reference tokens only — no hardcoded colors or fonts.
