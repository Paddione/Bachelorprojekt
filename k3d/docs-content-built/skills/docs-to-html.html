Plugin docs-to-html

Docs to HTML

Verwandelt lokale Dateien (Markdown, JSON, YAML, etc.) in ein interaktives, selbstaufrufendes HTML-Bundle für die Offline-Nutzung.

Auslöser: "Render diese Docs", "Bundle in HTML", "Mach einen One-Pager"
ℹ️
Offline-First: Das generierte HTML enthält alle Stile, Skripte und Assets (als Base64). Es benötigt keine CDN-Anbindung oder Internetverbindung.

Funktionsumfang

FeatureBeschreibung
Full-Text SearchIn-Memory Suche über alle gebündelten Dateien
Tag FilteringFilterung basierend auf Frontmatter und Verzeichnissen
Theme ToggleSchneller Wechsel zwischen Light- und Dark-Mode
Syntax HighlightingPrismJS für Code-Blöcke (Bash, JS, Python, etc.)
1 Input-Sammlung Dateien & Verzeichnisse

Gib Dateien, Globs oder ganze Ordner an. Unterstützte Formate: .md, .html, .json, .yaml, .txt, .log sowie Bilder.

/docs-to-html docs/superpowers/plans/ README.md
2 Layout-Wahl Heuristik oder Override

Das Layout wird automatisch basierend auf der Dateianzahl gewählt:

  • Single: < 4 Dateien (Linearer One-Pager)
  • Sidebar: 4 - 20 Dateien (Mit Navigationsmenü)
  • Grid: > 20 Dateien (Landing-Page mit Cards)
3 Build & Öffnen Node.js Build Script

Das Bundle wird in docs-html-bundles/ erstellt und automatisch im Browser geöffnet:

node scripts/build.mjs --config bundle.json
xdg-open docs-html-bundles/bundle-2026.html
