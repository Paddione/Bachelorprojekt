[ 03 — Angebote ]

Drei Formate. Eine Haltung.

SIGIL · KOPIE · PREIS

Drei Service-Archetypen mit eigener Sigille, Motto und Preisform — gemeinsames Vokabular, klar unterscheidbare Silhouette. Das Sigille ist kein Icon, sondern eine kleine editoriale Komposition: Hairline-Gitter, Messing-Strich, ein Anker.

[ 04 — Strip ]

Glaubwürdigkeit, knapp gesetzt.

STATS · VERFÜGBARKEIT

Die Stats-Strip-Komposition unter dem Hero. Vier Zahlen, ein Salbei-Puls, vier Slot-Pillen. Wird auch in Newsletter und Drucksachen wiederverwendet.

[ 05 — Palette ]

Zwölf Farben, jede mit Aufgabe.

WEB · DRUCK · GEMEINSAM

Tinte und Messing. Salbei für Verfügbarkeit, Stille-Blau für Tiefe, Tonrot nur bei Warnung. Keine Dekoration ohne Grund.

[ 06 — Stimme ]

Eine Zeile, ehrlich.

Die Quote-Card mit Brass-Halo, Newsreader-Italic, Brass-Avatar.

[ 07 — Ikonografie ]

Sechs Marken für die Begleitung.

STROKE · 1.4 · BRASS

Schmaler Stroke-Set für Offer-Karten, Newsletter und Drucksachen. Wenig Detail, klare Silhouette, Brass-Linie auf Tinte.

[ 08 — Oberflächen ]

Sechs Untergründe.

SURFACE · 240×160

Stimmungsproben für Hintergründe — wie Licht auf den Substraten der Marke liegt: Tinte, Messing-Halo, Stille-Blau, Hairline-Gitter, Duotone-Wash, Salbei-Puls.
