## Task 2 — Rewire sidebar nav

The sidebar "Sessions" link currently points to the (now-deleted) studio route.
Point it directly to the sessions list instead.

**File:** `website/src/components/admin/AdminSidebarNav.astro` (edit).

1. In the `navSections` array, find the item:

   ```ts
   { href: '/admin/coaching/studio', label: 'Sessions', icon: 'clipboard',
     matches: ['/admin/coaching/studio', '/admin/fragebogen'] },
   ```

2. Replace with:

   ```ts
   { href: '/admin/coaching/sessions', label: 'Sessions', icon: 'clipboard',
     matches: ['/admin/coaching/sessions', '/admin/fragebogen'] },
   ```

   The label stays „Sessions" — it now links to the sessions list which is the
   primary coaching workflow entry point. The `fragebogen` match is preserved.

**Verify Task 2:** `cd website && npx astro check` — no errors.
