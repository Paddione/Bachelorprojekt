# t001592 — Delta (Factory Floor agent configuration and sidekick settings)

## ADDED Requirements

### Requirement: Factory Floor MUST display provider badges for each station phase and open a drawer on click

The Factory Floor UI SHALL render a badge for each active phase indicating the assigned model/provider. Clicking on this badge SHALL open a drawer allowing administrators to manage provider entries for that phase.

#### Scenario: Clicking on the station phase badge opens the provider configuration drawer

- **GIVEN** an administrator is on the Factory Floor dashboard
- **WHEN** they click on a model/provider badge on one of the station columns
- **THEN** the provider configuration drawer for that phase is opened

### Requirement: Sidekick interface MUST expose global agent settings including context budget, spawn harness, lavish review, and kill switch

The Sidekick interface SHALL include an `'agent-settings'` view exposing controls for global orchestration parameters: context budget (`context_budget`), spawn harness toggle (`spawn-harness-toggle`), lavish delegation review toggle (`lavish-delegation-regel`), and global kill-switch (`kill-switch`).

#### Scenario: Navigating to agent settings shows orchestration parameters

- **GIVEN** an administrator has opened the Portal Sidekick
- **WHEN** they navigate to the `'agent-settings'` view
- **THEN** they can view and update the controls for context budget, spawn harness, lavish delegation, and the global kill-switch
